## 개요

문서를 어떻게 작성하는 지에 대한 가이드입니다.

## 문법

문장 마지막에 점은 무조건 찍습니다.

## 헤더

모든 문서는 `## 개요`로 시작합니다.
헤더 텍스트는 자연어 그대로 공백을 사용합니다.

`# 제목` 은 사용하지 않습니다.
제목의 경우엔 
```md
---
title: 제목
---

## 개요
```
로 작성합니다.

`#` 은 빌드 파일에 자동생성되기 때문.
raw 마크다운 파일로 렌더링된다할지라도 일관성을 위해 수동으로 작성 시 `#` 은 사용하지 않습니다.

## 텍스트 스타일링 금지 
볼드 처리를 할 때 헤더 제목 전체를 볼드 처리를 하지 않습니다.

```md
<!-- Bad -->
## **볼드 헤더**

<!-- Good -->
## 그냥 헤더
```

## 코드


코드 작성 시, 다른 언어거나 lua 와 관련없는 텍스트가 아니면 lua 로 작성합니다

```lua
-- code...
```

## 링크

문서 내 링크는 반드시 현재 문서를 기준으로 한 상대 경로(`../path/to/doc.md`)로 작성합니다.
저장소 루트 절대 경로나 외부 URL을 사용하는 방식은 허용하지 않습니다.
