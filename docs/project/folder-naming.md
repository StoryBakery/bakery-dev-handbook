## samples

`examples` 와 비슷하지만 같이 있는 경우
`examples` 는 한 주제에 대해서만 다루는 경향이 강하며
`samples` 는 보다 완성된 사용 사례를 보여준다.

```sh
examples/
  hello-world/
  streaming/
  custom-tool/

samples/
  customer-support-agent/
  ecommerce-application/
  multi-service-deployment/
```
